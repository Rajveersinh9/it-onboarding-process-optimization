# Security & Compliance Checklist

---

## Identity & Access Management

- MFA enabled
- Backup authentication method configured
- Least privilege permissions assigned
- Admin access removed from standard users

---

## Device Security

- FileVault encryption enabled
- Firewall active
- OS updated to latest patch version
- Auto-updates enabled

---

## Offboarding Procedure

When employee leaves:

- Disable Google Workspace account
- Remove Slack access
- Revoke VPN credentials
- Remove group memberships
- Deactivate SSO sessions
- Wipe device (factory reset)
- Update inventory status

---

## Audit Documentation

- Log account removal confirmation
- Verify MFA disabled
- Confirm device returned
- Document offboarding completion
