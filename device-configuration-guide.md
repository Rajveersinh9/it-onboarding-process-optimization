# macOS Device Configuration Guide

## Objective

Standardize macOS configuration to ensure consistency, security, and operational readiness for new hires.

---

## Step 1: Initial Setup

- Power on device
- Connect to secure Wi-Fi network
- Log in with administrative provisioning account

---

## Step 2: System Configuration

- Create standard user profile
- Enable automatic updates
- Configure screen lock timeout
- Set password complexity standards

---

## Step 3: Security Controls

- Enable FileVault disk encryption
- Confirm firewall enabled
- Configure automatic system updates
- Install endpoint security solution (simulated)

---

## Step 4: Application Installation

Install required applications:

- Google Chrome
- Slack
- Zoom
- VPN Client
- Company-approved utilities

---

## Step 5: Network Testing

- Verify Wi-Fi stability
- Confirm VPN connectivity (if applicable)
- Test internal system access

---

## Step 6: Final Validation

- Log in as end-user
- Confirm email + Slack functionality
- Validate MFA prompt working
- Document configuration in onboarding ticket
